# CLAUDE.md 输出模板

> 从 SKILL.md 拆分。生成的 CLAUDE.md 建议包含以下节（按优先级排序，缺失某类则跳过）。

```markdown
# CLAUDE.md — [项目名]

> [如为非软件项目：项目类型声明——1行]

## Agent 边界
- 可派：...
- 禁止：...

## 环境与命令
- 运行环境：...
- 关键命令：[具体可粘贴的 CLI 命令 + 用途]

## 停止条件
- [行为禁令列表]

## 已知坑位
- [易误解概念 + 易误操作]

## 更新协议
- [触发条件 + 更新后必做]
```

> 以上为建议结构，实际章节依脆弱性测试结果增减。**不上"快速恢复/目录地图/核心资产/版本历史"——这些是可推导信息或已有其他事实源。**
