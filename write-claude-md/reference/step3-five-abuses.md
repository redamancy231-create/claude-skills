# Step 3：五滥检查

> 从 SKILL.md 拆分。清除 CLAUDE.md 中 Agent 可通过其他途径获取或对决策无用的信息。

扫描所有内容，以下五类**出现即砍**：

- `[ ]` **可推导信息**：`ls`/`Glob`/读构建文件就能获取的 → 砍
- `[ ]` **通用惯例**：语言标准规范（PEP8/DRY/"写整洁代码"）→ 砍
- `[ ]` **易变信息**：版本号（引用 `VERSION` 文件即可）、文件 KB 数、模型/工具白名单 → 砍或指针化
- `[ ]` **文件详述**：对每个文件的罗列描述 → 砍（Glob + README 负责）
- `[ ]` **空泛建议**："遵循最佳实践"等不言自明的原则 → 砍
