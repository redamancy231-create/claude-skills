# 非软件项目额外检查

> 从 SKILL.md 拆分。方法论文档/研究/内容/知识库类项目须额外确认。

- `[ ]` **项目类型声明**：第一行声明"本仓库是 X 项目（非软件），核心交付物是 Y"
- `[ ]` **术语定义**：操作动词（试跑/验证/测试/同步）是否在项目语境下有非默认含义
- `[ ]` **节号稳定性**：如果 CLAUDE.md 引用另一文档的节号，须加"编辑后验证节号有效性"
- `[ ]` **追加式日志**：如果 `project_status.md` 是追加式（最新在末尾），须标注"从末尾读"
- `[ ]` **Monorepo**：多子项目仓库优先每个子项目各写 CLAUDE.md，根级别只放跨项目共享约束
- `[ ]` **多语言同步**：如有 `en/CLAUDE.md` 等译文，主语言 CLAUDE.md 修改后须标记译文滞后
