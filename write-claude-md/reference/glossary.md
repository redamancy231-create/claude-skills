# 术语定义

> 从 SKILL.md 拆分。

| 术语 | 定义 | 反例 |
|------|------|------|
| **白名单指针化** | CLAUDE.md 不列具体模型/工具名，只写"见 X 文件" | `已验证可用：GPT-5.5 / Qwen / Kimi` ← 易变，应移到 project_status.md |
| **可推导信息** | Agent 通过 `ls`/`Glob`/读构建文件就能获取的内容 | 目录地图、文件列表、依赖版本号 |
| **易变信息** | 随版本/时间/环境变化的内容，维护会制造漂移 | 文件 KB 数、模型白名单、更新日期 |
| **重要项目** | 复杂度 ≥3 子系统，或误操作后果不可逆 | 个人 scratch 项目通常不算 |
