# Step 2：五缺检查

> 从 SKILL.md 拆分。确保 CLAUDE.md 覆盖 Agent 必需的五大信息类别。

扫描保留内容，以下五类**必须至少各有一条**。任一为零则从 Step 0 采集的证据中补充：

- `[ ]` **关键命令**：构建/测试/发布前检查的具体 CLI 命令（不可从代码推导）
- `[ ]` **定制规范**：与语言/框架默认不同的约定（如"用 pnpm 不用 npm"）
- `[ ]` **架构约束**：不可修改的目录、禁止操作、设计不变量
- `[ ]` **环境要求**：非标准工具链、必需环境变量、平台特殊配置
- `[ ]` **避坑指南**：已知陷阱、易误解概念、反复出现的错误模式
