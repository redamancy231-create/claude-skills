# 门2：预登记冻结

> 从 SKILL.md 拆分。仅在门1 Decision=GO 后执行。

## 门1/门2 数据边界（关键）

门1 跑 cheap test 用过的数据**只算探索性证据**；门2 若要宣称「验证」，**必须冻结新的 holdout / 时间切片 / 外部数据**，不许把看过的数据包装成 confirmatory。

## 预登记项目

- **一句话命题** + **假设**：H1（主）/ **H0 必须如实报告**（null 也是结论，不得事后改协议合理化）。
- **主指标 + 噪声底**：任何分差先减噪声底；点估计不可信，要 CI。
- **kill 判据 / 全量门槛**：定死，非建议。
- **时序铁律**：冻结协议 → 抽样/取数 → 定评分锚点 → 跑 → 评判。
- **冻结后铁律**：协议/指标/kill 判据**不得因数据结果而改**；只能记 **amendment** 并说明**非因结果**。
- 登记**已知残余混杂 + 缓解**与**利益冲突**。
