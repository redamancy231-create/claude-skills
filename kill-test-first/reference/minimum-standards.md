# 关键概念的最低合格标准

> 从 SKILL.md 拆分。防"填了表但不合格"。

- **最强反方**：针对核心机制的理论 kill，非边角、非 strawman。
- **先验对标**：① 术语三角(≥3 角度、≥2 类来源) ② 具名先例落到可验证链接(AI 只算候选) ③ 差异比到 核心问题/输出物/评估指标 三项 ④ 搜不到只能写「检索不足 + negative_scope」、不许写「未发现先例」。⑤ **来源验证(P2新增)：逐条标注 verified/unverified/ai-generated，verified=0 不得 GO。** **只写一句"搜过、没近邻" = 不合格 → 标 incomplete 重搜，不得 GO**。
- **朴素基线**：不使用核心新假设的**最强**简单替代（如直接动量 / 货基+指数），**不是故意弱的稻草人**。
- **噪声底**：同一对象独立重测 ≥3 次的方差。
- **独立角色**：未参与构思与设计过程。
- **样本外**：切分规则明确、无重叠、候选的「未来」早于目标。
