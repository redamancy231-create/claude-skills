# Next Steps 标准化格式

> 从 SKILL.md §2.3 提取。生成 Next Steps 时套用此模板。

## 格式

```
操作动词 + 具体文件/命令(绝对路径) + 优先级(P0/P1/P2) + 依赖(等什么完成后才能做)
```

## 优先级定义

| 级别 | 定义 | 示例 |
|------|------|------|
| **P0** | 阻塞下一阶段/有时限/不进下一步项目废 | 验证结果槽模板格式 |
| **P1** | 本阶段应完成但不阻塞下一阶段启动 | 更新因子权重 |
| **P2** | 锦上添花/可被后续会话自主判断 | 补充单元测试 |

## 示例

```
运行 projects/my-project/test.py 验证result.csv模板 → P0 → 等数据更新后
更新 strategy.py 因子权重 → P1 → 等V3.7回测完成后
重构 utils.py 异常处理逻辑 → P2 → 无依赖
```
